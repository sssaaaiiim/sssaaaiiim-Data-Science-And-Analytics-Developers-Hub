# Data Science And Analytics Developers Hub

📂 **Repository Structure**  
This repository contains the complete set of projects developed during my internship at DeveloperHubs. Each project focuses on solving a real-world problem using data science and machine learning techniques.

## 1. Bank Marketing Dataset
Objective: Predict whether a client will subscribe to a term deposit.  
Techniques: Classification, managing imbalanced classes, categorical data encoding.

## 2. Customer Churn Dataset
Objective: Identify customers who are likely to cancel their service or subscription.  
Techniques: Binary classification, feature scaling, exploratory visualizations to identify churn indicators.

## 3. Insurance Dataset
Objective: Predict individual medical costs billed by health insurance based on patient demographics.  
Key Features: Age, BMI, Smoking Status, Region.  
Techniques: Linear regression, handling non-linear interactions, numerical imputation, scatter plot visualizations.

## 4. Iris Dataset
Objective: Classify iris plants into three distinct species based on sepal and petal measurements.  
Techniques: Multiclass classification, standard scaling, fundamental data modeling.

## 5. Loan Dataset
Objective: Predict the likelihood of a borrower defaulting on a loan.  
Key Features: Credit Score, Interest Rate, Debt-to-Income Ratio, Loan Term.  
Techniques: Random Forest and Decision Tree classifiers, ColumnTransformer pipelines, confusion matrix evaluation.

## 🛠️ Technologies & Tools Used
- **Data Manipulation & Analysis:** pandas, numpy  
- **Machine Learning Framework:** scikit-learn (Pipelines, ColumnTransformers, Ensembles)  
- **Data Visualization:** matplotlib, seaborn  


This repository demonstrates the end-to-end application of data science workflows — from data exploration and preprocessing to model training, evaluation, and visualization — across multiple domains.
